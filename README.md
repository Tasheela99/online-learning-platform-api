# Online Learning Platform Api

## 01.Install nodemon
    install the nodemon

    npm install -g nodemon

## 02.Run npm install
    type npm install to install the dependencies

    npm install

## 03.Run The application
    tyoe nodemon in the integrated terminal to run the api

    nodemon
